# Muhammad Usman - Interactive Resume

A modern, interactive resume website for Muhammad Usman, an E-Commerce Professional specializing in Xiaomi products.

## Features

- **Interactive Elements**: Smooth navigation, theme toggle, and responsive design
- **Modern Design**: Frosted glass navigation, card-based layout, subtle animations
- **User-Friendly Intro**: Guided tour of website features for first-time visitors
- **Responsive**: Fully responsive design for all devices
- **Print Optimized**: Clean print layout for physical resumes
- **Dark/Light Mode**: Toggle between light and dark themes
